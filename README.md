# Interactive CLI to start CSMA/SSMA projects.

## Quick Start

Rune the following command to create your project.

```bash
npx -y github:yagaltd/csma-ssma-cli \
  --template-source github \
  --csma-repo yagaltd/CSMA \
  --ssma-repo yagaltd/SSMA
```
